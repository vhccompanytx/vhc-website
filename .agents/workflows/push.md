---
description: How to push changes to the VHC Website Git repository
---

// turbo-all
1. Stage all changes:
```powershell
& "C:\Program Files\Git\cmd\git.exe" add .
```

2. Commit changes with a descriptive message:
```powershell
& "C:\Program Files\Git\cmd\git.exe" commit -m "Update from Antigravity"
```

3. Push to the remote repository:
```powershell
& "C:\Program Files\Git\cmd\git.exe" push origin main
```
